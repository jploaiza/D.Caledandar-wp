# Reservas Terapia Plugin

## Webhook Configuration

### Twilio WhatsApp Webhook
To enable secure communication with Twilio:
1. Go to [Twilio Console](https://console.twilio.com/).
2. Navigate to your WhatsApp Sandbox or Sender configuration.
3. specific the "When a message comes in" URL to: `https://yoursite.com/wp-json/reservas-terapia/v1/twilio-webhook`.
4. Ensure HTTP Method is set to `POST`.
5. Get your **Auth Token** from the dashboard and save it in the plugin settings (Integrations > Twilio).

### Zoom Webhook
To enable real-time updates from Zoom:
1. Go to [Zoom Marketplace](https://marketplace.zoom.us/).
2. Create or manage your OAuth App.
3. Go to **Feature > Event Subscriptions**.
4. Enable Event Subscriptions and add a new subscription.
5. Set the Event notification endpoint URL to: `https://yoursite.com/rt/v1/zoom/webhook` (Note: verify the route prefix in your setup).
6. Copy the **Secret Token** generated by Zoom.
7. Paste this token in the plugin settings (Integrations > Zoom > Webhook Secret Token).
8. Subscribe to the following events:
   - Meeting: `meeting.started`, `meeting.ended`
   - Participant: `participant.joined`, `participant.left`
